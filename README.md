# Roger Scream Button

Mini projeto web com HTML, CSS e JavaScript.

## Sobre
Este projeto reproduz um trecho de áudio ao clicar em um botão, com efeitos visuais inspirados em palco de rock.

## Funcionalidades
- reprodução de áudio ao clique
- efeito de flash
- animação no botão
- layout centralizado e responsivo

## Tecnologias usadas
- HTML5
- CSS3
- JavaScript
